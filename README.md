# Messenger API
An API for creating automatic tasks :
## Features
- Invite someone to a groupchat conversation
- Read or send messages on a specific conversation
- See the unread messages
- Generate a temporary Facebook account

This API was created to implement chatbots in a group conversation of Messenger since this feature has not been implemented yet.
